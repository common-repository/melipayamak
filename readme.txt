<b>Melipayamak SMS Plugin for WordPress and Woocommerce</b>

<p>This plugin offers SMS automation and marketing solutions to Iranians. It works with different extensions of WordPress including Woocommerce, Easy Digital Downloads and Form Creator.</p>

<p>By using this extension, it’s possible to send your customers and subscribers, automatic SMS notifications on different occasions (For example, when a new post is uploaded or a product, sent.) You can also use this plugin to send SMS manually to your users or any other custom number. The extension is also capable of creating a sign up form on your website for SMS subscription.</p>

<p>Naturally the language of this extension is in Persian. So, it wouldn’t be of a great help to speakers of other languages.</p>

<p>The other important thing is that the extension only works with www.melipayamak.com SMS Web service. Therefore, in order to use it, you have to buy an SMS panel from the website.</p>

== Installation ==
1. Install the latest version of SourceGaurdian on your server.
2. Install MeliPayamak extension on your WordPress CMS.
3. Now, go to the MeliPayamak section which is added to WordPress menu, choose "تنظیمات", then "تنظیمات کاربری" and enter your melipayamak.com account information and the number you want to use when you're sending an SMS.
